# Security Policy

## Supported Versions

| Version | Supported |
|---------|-----------|
| 26.05.x | ✅ Yes |
| < 26.05 | ❌ No |

## Reporting a Vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Report privately via [GitHub Security Advisories](https://github.com/thisismyurl/easy-popular-posts/security/advisories/new) or email **security@thisismyurl.com**.

Include:
- Plugin version affected
- WordPress version
- Steps to reproduce
- Potential impact

You will receive a response within **72 hours**. Critical issues are patched as soon as reasonably possible and disclosed responsibly after a fix is available.
